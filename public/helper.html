🔧 京东抓包助手

两种方式帮你自动提取 API 数据

方式 1：书签工具（推荐）

将下面的链接拖到浏览器书签栏：

📌 拖我到书签栏

使用方法：

  1. 将上面的链接拖到书签栏
  2. 打开京东活动页面
  3. 点击书签
  4. 点击领券按钮
  5. 按 F12 查看控制台，会显示抓到的请求

方式 2：控制台脚本

在京东活动页面按 F12，粘贴以下代码到控制台：

// 监听所有请求 const originalFetch = window.fetch; window.fetch = function(...args) { const url = args[0]; const options = args[1] || {}; if (url.includes('api.m.jd.com')) { console.log('%c捕获到 API 请求！', 'color: #4CAF50; font-size: 16px; font-weight: bold'); console.log('URL:', url); console.log('Method:', options.method || 'GET'); console.log('Headers:', options.headers); console.log('Body:', options.body); console.log('---完整数据---'); console.log({ url: url, method: options.method || 'POST', headers: options.headers, body: options.body }); } return originalFetch.apply(this, args); }; console.log('✅ 抓包脚本已启动！现在点击领券按钮...');

方式 3：一键自动填充

抓到数据后，使用这个脚本自动填充到工具：

// 在控制台执行，自动打开工具并填充数据 const data = { cookie: document.cookie, body: '粘贴抓到的 body 数据' }; // 打开工具页面 const win = window.open('你的工具地址/jd.html', '_blank'); // 等待页面加载后填充 setTimeout(() => { win.document.getElementById('cookie').value = data.cookie; win.document.getElementById('body').value = data.body; }, 1000);

📱 手机端自动抓包

推荐工具：
