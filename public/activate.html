🔓 京东按钮点亮工具

一键激活灰色领券按钮，提前抓包获取请求数据

📌 拖拽书签到书签栏

最简单的使用方式：保存书签，一键点亮！

🔓 一键点亮
⚠️ 用鼠标拖拽（不是点击）到浏览器顶部的书签栏
💡 如果看不到书签栏，按 Ctrl+Shift+B（Mac: Cmd+Shift+B）
✨ 工具特点：
1. 💨 快速：一键激活所有灰色按钮
2. 🎯 精准：智能识别领券按钮
3. 🔒 安全：纯前端脚本，不上传任何数据
4. 📱 通用：支持所有京东活动页面

📚 使用步骤

  1. 保存书签 拖拽上方的"🔓 一键点亮"按钮到浏览器书签栏
  2. 打开京东活动页面 访问任何有灰色领券按钮的京东活动页面
  3. 点击书签 点击书签栏的"🔓 一键点亮"，等待弹窗提示
  4. 点击领取按钮 现在灰色按钮已经点亮，可以点击了

💻 方法2：手动运行脚本

如果书签不方便，可以在控制台手动运行脚本

// 京东按钮点亮脚本
(function() {
    let count = 0;

    // 1. 激活所有可能的按钮元素
    const selectors = [
        'button',
        '.btn',
        '.button',
        '[class*="receive"]',
        '[class*="coupon"]',
        'a[href*="receive"]'
    ];

    selectors.forEach(sel => {
        document.querySelectorAll(sel).forEach(el => {
            el.disabled = false;
            el.removeAttribute('disabled');
            el.classList.remove('disabled', 'gray', 'grey', 'btn-disabled');
            el.style.cssText += 'pointer-events:auto!important;opacity:1!important;cursor:pointer!important;filter:none!important;';
            count++;
        });
    });

    // 2. 查找包含领取文字的元素
    document.querySelectorAll('*').forEach(el => {
        const text = el.innerText;
        if (text && text.length < 30 && /领取|立即|抢购|马上/.test(text)) {
            el.disabled = false;
            el.removeAttribute('disabled');
            el.classList.remove('disabled', 'gray', 'grey');
            el.style.cssText += 'pointer-events:auto!important;opacity:1!important;cursor:pointer!important;';
            console.log('✓ 激活:', text.trim().substring(0, 15));
            count++;
        }
    });

    console.log('\n✅ 已点亮', count, '个按钮');
    alert(`✅ 已点亮 ${count} 个按钮！\n\n💡 现在可以点击领取按钮了`);
})();

💡 使用方法：在京东页面按 F12 打开控制台，粘贴上面的代码，按回车运行

⚠️ 重要说明：
1. 点亮按钮 ≠ 真的能领券，服务器会检查时间和库存
2. 主要用于提前抓包，获取请求参数
3. 配合 h5st 调试工具 使用效果更好
4. 仅供学习研究使用，请勿用于违规操作

❓ 常见问题

Q: 点亮后按钮还是点不了？

A: 可能页面有 JavaScript 检查。尝试：
1. 刷新页面后再运行脚本
2. 使用浏览器的"检查元素"直接修改按钮
3. 尝试修改系统时间（高级）

Q: 点亮后点击按钮还是提示"未开始"？

A: 这是正常的！服务器端会检查时间。点亮按钮的主要目的是让你能点击按钮，从而触发网络请求，方便抓包获取 h5st 签名等参数。

Q: 如何配合 h5st 调试工具使用？

A: 完整流程：
1. 先运行 h5st 监听脚本
2. 再运行本工具点亮按钮
3. 点击按钮后，h5st 脚本会自动捕获签名
4. 在控制台查看捕获的数据

Q: 书签拖不进去？

A: 确保：
1. 书签栏已显示（按 Ctrl+Shift+B）
2. 用鼠标按住拖拽，不是点击
3. 或者直接使用"方法2"的手动脚本
